# Strength Tools

A small mobile-friendly web toolkit.

## Apps

- **Plate Loader** — calculate the plates needed on each side of a bar.
- **Performance Profile** — grade squat, bench press, deadlift, one-mile run, and waist circumference against documented reference data.

Open `index.html` to launch the tools hub.

## Performance Profile sources

See [`performance-grader/SOURCES.md`](performance-grader/SOURCES.md) for licenses, formulas, thresholds, and limitations.

## Hosting

The repository is static HTML/CSS/JavaScript and can be hosted directly with GitHub Pages.
